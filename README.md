# google-snake-game

A simple Snake game inspired by Google Snake with additional features.

## How to Play

- Use the arrow keys to control the snake.
- Eat the red food to grow and increase your score.
- Avoid hitting the walls or yourself (loses a life).
- Press F to fire a bullet in the direction the snake is moving.
- Bullet hitting wall: +67 lives
- Bullet hitting food: removes that food (-1 food count)

## Running the Game

Open `index.html` in your web browser to start playing.

If you have a local server setup (like Live Server extension in VS Code), you can serve the files for a better experience.

## Features

- Classic Snake gameplay with lives system
- Multiple food items
- Bullet firing mechanic
- Score and food count tracking
